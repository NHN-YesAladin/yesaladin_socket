# yesaladin_socket

YesAladin socket은 shop 서버와 클라이언트간 Websocket 통신을 지원하는 Websocket Server입니다.

## Getting Started

```bash
./mvnw spring-boot:run
```

## Features

### [@김홍대](https://github.com/mongmeo-dev)

- Spring Websocket, SockJS, STOMP를 사용하여 웹소켓 서버 구축

- Feature: 기능 개발 단계
- Hot-Fix: Merge 후 발생한 버그 및 수정 사항 반영 시 사용
